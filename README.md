## Description

## Development Build
Execute command:
```
npm run dev
```

To run the test execute the command:
```
npm run test
```

## Production Build

1. Create a __.env__ file.

2. Copy the variables from file __example.env__ into __.env__ .

3. Execute command: 
```
npm run start
```
